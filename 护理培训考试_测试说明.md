# 护理培训及考试系统 - 自动化测试说明

## 📋 概述

基于Playwright + pytest框架的自动化测试脚本，覆盖护理培训及考试系统的核心功能模块。

## 🎯 测试覆盖模块

| 模块 | 测试用例数 | 功能点 |
|------|-----------|--------|
| 登录模块 | 3个 | 正常登录、错误密码、空字段验证 |
| 视频学习 | 4个 | 播放、暂停、进度保存、全屏 |
| 文件上传 | 3个 | PDF上传、进度显示、取消上传 |
| 考试模块 | 5个 | 开始考试、单选、多选、提交、超时 |
| 满意度调查 | 4个 | 滑块、单选、多选、提交 |
| 课程列表 | 5个 | 加载、搜索、筛选、详情、分页 |

**总计：24个测试用例**

## 🛠️ 环境准备

### 1. 安装Python依赖

```bash
pip install pytest pytest-asyncio playwright
playwright install
```

### 2. 准备测试文件

```bash
mkdir -p test_files
# 放入测试用的PDF文件
# 放入测试用的大视频文件
```

### 3. 配置测试环境

修改脚本中的配置：
```python
BASE_URL = "http://localhost:8080"  # 你的应用地址
USERNAME = "test_user"              # 测试账号
PASSWORD = "test_pass"              # 测试密码
```

## 🚀 运行测试

### 方式1：直接运行（推荐）

```bash
python 护理培训考试_自动化测试.py
```

### 方式2：使用pytest

```bash
# 运行所有测试
pytest 护理培训考试_自动化测试.py -v

# 运行指定模块
pytest 护理培训考试_自动化测试.py::test_login -v
pytest 护理培训考试_自动化测试.py::test_video -v
pytest 护理培训考试_自动化测试.py::test_exam -v

# 生成HTML报告
pytest 护理培训考试_自动化测试.py --html=report.html --self-contained-html
```

### 方式3：使用pytest-xdist并行运行

```bash
pip install pytest-xdist
pytest 护理培训考试_自动化测试.py -n auto
```

## 📊 CI/CD集成

### GitHub Actions示例

```yaml
name: Automated Tests

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Set up Python
        uses: actions/setup-python@v2
        with:
          python-version: '3.9'
      - name: Install dependencies
        run: |
          pip install pytest pytest-asyncio playwright
          playwright install
      - name: Run tests
        run: pytest 护理培训考试_自动化测试.py -v --tb=short
      - name: Upload screenshots
        uses: actions/upload-artifact@v2
        if: failure()
        with:
          name: screenshots
          path: screenshots/
```
