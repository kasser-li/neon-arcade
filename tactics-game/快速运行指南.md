# 快速修复版 - 1 分钟运行

## 🚀 超简单步骤

### 1. 打开 Blender
### 2. 删除默认立方体（选中按 X）
### 3. 按 Shift+F11 打开脚本编辑器
### 4. 打开 `blender_knight.py`
### 5. 点击右上角 ▶ 运行

---

## ❌ 如果还报错

说明 Blender 版本太老或太新，试试这个**备用方案**：

### 方案 A：使用简化版脚本

```python
import bpy

# 清除场景
bpy.ops.object.select_all(action='SELECT')
bpy.ops.object.delete()

# 创建简单骑士（用基础几何体）
# 身体
bpy.ops.mesh.primitive_cylinder_add(radius=0.3, depth=1.2, location=(0, 1.5, 0))
body = bpy.context.active_object
body.name = "Body"

# 头
bpy.ops.mesh.primitive_sphere_add(radius=0.25, location=(0, 2.3, 0))
head = bpy.context.active_object
head.name = "Head"

# 创建材质
mat = bpy.data.materials.new(name="Armor")
mat.diffuse_color = (0.8, 0.8, 0.8, 1)
body.data.materials.append(mat)
head.data.materials.append(mat)

print("✅ 简化版骑士创建完成！")
```

复制上面代码，粘贴到 Blender 脚本编辑器运行。

### 方案 B：手动建模（5 分钟）

1. **身体**：添加圆柱体（Shift+A → Mesh → Cylinder）
2. **头**：添加球体
3. **四肢**：添加圆柱体
4. **上色**：右侧材质面板添加新材质
