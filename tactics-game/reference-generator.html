📖 如何在 Blender 中使用

  1. 点击"下载参考图"保存为 PNG
  2. 打开 Blender，按 Shift + A → Image → Reference
  3. 选择下载的 PNG 文件
  4. 按 N 打开侧边栏，调整参考图位置
  5. 在正交视图 (按 1/3/7) 中照着描图建模
  6. 导出时删除参考图

📐 比例说明

  1. 头身比：约 1:7.5（写实风格）
  2. 肩宽：约 2.5 个头宽
  3. 腿长：约占身高 1/2
  4. 网格单位：1 Blender 单位 = 1 米
