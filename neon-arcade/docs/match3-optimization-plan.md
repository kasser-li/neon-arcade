# 霓虹消消乐优化预案

## 当前状态

✅ 已完成：
- 无尽模式（基础版）
- 闯关模式（10关）
- 基础特效系统（火箭、炸弹、彩虹球）
- 连击系统
- 本地存档

---

## 优化方向

### 1. 道具系统 ⭐⭐⭐

**前置道具（开局前使用）：**
- 🔨 锤子 - 消除任意一个元素
- 🎲 刷新 - 重新排列棋盘
- ➕ 额外步数 - 增加3步
- 🌈 开局彩虹球 - 开局自带一个彩虹球

**局内道具（游戏中使用）：**
- 💣 小炸弹 - 消除3x3区域
- 🎯 提示 - 显示可移动位置
- ⏱️ 时间冻结 - 无尽模式暂停倒计时

**实现建议：**
```typescript
// stores/match3Shop.ts
interface PowerUp {
  id: string
  name: string
  icon: string
  description: string
  price: number  // 用游戏内金币
  type: 'pre' | 'in-game'
  effect: () => void
}
```

---

### 2. 商店系统 ⭐⭐⭐

**货币系统：**
- 💰 金币 - 通过游戏获得，用于购买道具
- 💎 钻石 - 稀有货币，可通过成就或充值获得

**商品分类：**
- 道具包
- 头像框
- 主题皮肤（糖果、宝石、水果等）
- 特效升级（更炫酷的消除动画）

**实现建议：**
```typescript
// 金币获取途径
- 每消除1个元素：+1金币
- 完成关卡：基础奖励 + 剩余步数奖励
- 连击奖励：连击数 x 5金币
- 每日登录：100金币
```

---

### 3. 每日挑战 ⭐⭐

**每日特殊关卡：**
- 每天生成一个特殊配置的关卡
- 限时24小时
- 完成后获得丰厚奖励
- 全球排行榜（当日最高分）

**挑战类型：**
- 限时挑战（60秒得最高分）
- 步数挑战（10步内完成目标）
- 收集挑战（收集100个指定元素）

---

### 4. 关卡扩展 ⭐⭐⭐

**新增关卡元素：**

| 元素 | 说明 | 清除方式 |
|-----|------|---------|
| 🧊 单层冰 | 覆盖在元素上 | 消除下方元素 |
| 🧊🧊 双层冰 | 需要消除两次 | 消除两次 |
| 🕸️ 蛛网 | 固定元素无法移动 | 匹配消除 |
| 🧱 石板 | 阻挡掉落 | 消除周围元素 |
| 🎁 礼物盒 | 消除后随机生成道具 | 匹配消除 |
| 💣 定时炸弹 | 步数倒计时，爆炸失败 | 消除或道具清除 |

**新关卡类型：**
- 收集关卡 - 收集指定数量的特定元素
- 限时关卡 - 时间限制内完成目标
- 生存关卡 - 尽可能多得分，步数用完后可继续（消耗金币）
- Boss关卡 - 消除特定元素对Boss造成伤害

**背景音乐：**

### 7. 动画优化 ⭐⭐

**消除动画：**

**特效升级：**

**实现建议：**
```css
/* 使用CSS动画 + Web Animations API */
.cell.is-matched {
  animation: explode 0.3s ease-out forwards;
}

@keyframes explode {
  0% { transform: scale(1); opacity: 1; }
  50% { transform: scale(1.3); opacity: 0.8; }
  100% { transform: scale(0); opacity: 0; }
}
```

## 技术实现注意

1. **数据持久化** - 使用 localStorage + IndexedDB（大量数据时）
2. **性能优化** - 动画使用 CSS transform，避免频繁重排
3. **响应式** - 确保移动端体验流畅
4. **离线可用** - 所有功能支持离线游玩，联网后同步数据
