## 💡 技术要点

### AGV 状态机
```
空闲 → 工作中 (任务分配)
工作中 → 充电中 (电量<15%)
充电中 → 空闲 (电量=100%)
工作中 → 空闲 (任务完成/取消)
```

### 模拟逻辑
- 每秒更新 AGV 位置（随机偏移）
- 工作时电量 -0.1%/秒
- 充电时电量 +0.5%/秒
- 位置使用 Leaflet marker 动态更新

### 路径规划
- 简化版：直线连接起点终点
- 可扩展：A*算法、Dijkstra 算法
- 支持路径冲突检测（未来）
