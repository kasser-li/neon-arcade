## 🎴 卡牌战旗 v2 护甲机制修复 (15:25)

### Bug 发现与修复
**问题**: 用户使用防御牌获得护甲后，敌人攻击时仍然直接扣血

**根因**: `endTurn()` 函数中护甲在敌人攻击前就被清零了
```javascript
// 错误代码
function endTurn(){
    gameState.player.block=0;  // ❌ 太早清零
    setTimeout(enemyAction,800);
}
```

**修复**: 护甲应该在下一回合开始时清零，而不是结束回合时
- 结束回合 → 护甲保留
- 敌人攻击 → 护甲先格挡
- 新回合开始 → 护甲刷新清零

### 游戏机制确认
护甲逻辑符合杀戮尖塔规则:
- 护甲只在**当前回合 + 敌人攻击阶段**有效
- 下回合开始时刷新 (清零)
- 不是永久叠加的
