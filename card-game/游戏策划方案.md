# 🎴 卡牌战旗 - 游戏策划方案

> 杀戮尖塔类卡牌 Roguelike 游戏

---

## 📋 游戏概述

### 核心概念
**爬塔 + 卡牌构建 + Roguelike**
- 玩家从塔底向上挑战，每层选择不同路径
- 通过战斗获得新卡牌，构建独特卡组
- 永久死亡，每次冒险都是新的开始

### 游戏循环
```
选择路径 → 遭遇事件 → 战斗/奖励 → 强化卡组 → BOSS 战 → 下一层
```

### 目标平台
- Web 浏览器（HTML5 + Canvas）
- 后续可扩展到移动端

---

## 🎮 核心玩法

### 1. 战斗系统

#### 回合制卡牌战斗
- **玩家回合**: 抽牌 → 出牌 → 结束回合
- **敌人回合**: 显示意图 → 执行行动
- **能量系统**: 每回合固定能量，卡牌消耗能量

#### 战斗界面布局
```
┌─────────────────────────────────────┐
│  敌人 (血条 + 意图图标)              │
│                                     │
│  玩家 (血条 + 能量 + 护甲)           │
│                                     │
│  ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐    │
│  │卡牌│ │卡牌│ │卡牌│ │卡牌│ │卡牌│    │
│  └───┘ └───┘ └───┘ └───┘ └───┘    │
│                                     │
│  [结束回合]  [取消]                  │
└─────────────────────────────────────┘
```

#### 意图系统
敌人行动前显示意图图标：
- ⚔️ **攻击**: 显示伤害数值
- 🛡️ **格挡**: 获得护甲
- 💪 **强化**: 增益效果
- 💀 **Debuff**: 负面状态

### 2. 卡牌系统

#### 卡牌品质
| 品质 | 颜色 | 获取概率 |
|------|------|----------|
| 普通 | 白色 | 60% |
| 稀有 | 蓝色 | 30% |
| 史诗 | 紫色 | 9% |
| 传说 | 金色 | 1% |

#### 卡牌类型
| 类型 | 图标 | 说明 |
|------|------|------|
| 攻击 | ⚔️ | 造成伤害 |
| 技能 | 🛡️ | 获得护甲/效果 |
| 能力 | 💪 | 持续增益 |
| 状态 | ⚡ | 特殊效果 |

#### 初始卡组示例（10 张）
```
打击 x4     (造成 6 点伤害)
防御 x4     (获得 5 点护甲)
痛击 x1     (造成 8 点伤害，施加 1 层虚弱)
格挡 x1     (获得 8 点护甲)
```

## 🎁 奖励系统

### 战斗胜利奖励
- **基础金币**: 20-50 金币
- **卡牌奖励**: 战斗结束后从 3 张随机卡牌中选择 1 张
- **遗物掉落**: BOSS 战必掉遗物

### 卡牌获取机制
#### 通用卡牌
所有角色都可获得的卡牌，占比 70%

#### 职业专属卡牌
仅特定角色可获得，占比 30%
- **骑士专属**: 盾牌相关、格挡反击、战吼类
- **法师专属**: 法术伤害、法力回复、元素系
- **刺客专属**: 暴击、连击、毒素系

### 奖励选择界面
```
┌─────────────────────────────────────┐
│        选择一张卡牌加入卡组          │
│                                     │
│  ┌───────┐  ┌───────┐  ┌───────┐   │
│  │ 重击  │  │ 格挡  │  │ 狂热  │   │
│  │ ⚔️稀有 │  │ 🛡️普通│  │ 💪史诗│   │
│  │12 伤害 │  │10 护甲 │  │ +1 能量 │   │
│  └───────┘  └───────┘  └───────┘   │
│                                     │
│        [确认]    [跳过]             │
└─────────────────────────────────────┘
```

### 每日首胜奖励
- 首次通关额外获得 1 张稀有卡牌
- 解锁新职业专属卡牌

## 👾 敌人设计

### 第 1 层敌人

#### 史莱姆
- **HP**: 40
- **行动模式**: 攻击 (6) → 分裂 → 攻击 (4) ×2
- **特殊**: 死亡时分裂为 2 个小史莱姆

#### 盗贼
- **HP**: 50
- **行动模式**: 偷窃 → 攻击 (8) → 闪避
- **特殊**: 有概率闪避攻击

### 第 2 层精英

#### 黑暗骑士
- **HP**: 100
- **行动模式**: 重击 (12) → 蓄力 → 致命一击 (20)
- **特殊**: 每 3 回合一次高伤害攻击

### 第 3 层 BOSS

#### 巨龙
- **HP**: 200
- **行动模式**: 
  - 普通攻击 (10)
  - 龙息 (全体 15 点伤害)
  - 恐惧 (施加 2 层脆弱)
- **阶段转换**: 50% HP 时进入狂暴

## 🎲 Roguelike 元素

### 永久死亡
- 每次冒险独立
- 死亡后保留解锁内容

### 进度解锁
- **新角色**: 通关解锁
- **新卡牌**: 击败特定 BOSS 解锁
- **新遗物**: 探索发现后加入卡池

### 遗物系统
| 遗物 | 效果 |
|------|------|
| 能量宝石 | 战斗开始时 +1 能量 |
| 吸血之刃 | 攻击时回复 1HP |
| 时间沙漏 | 每 3 回合免费使用一张牌 |
| 幸运四叶草 | 稀有卡牌出现率 +10% |
