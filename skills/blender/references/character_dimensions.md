# 角色建模比例参考

## 📐 写实风格（7.5 头身）

### 整体比例

```
总身高：1.75 - 1.85 米
头高：约 0.23 - 0.25 米
头身比：1 : 7.5
```

### 身体各部分比例

| 部位 | 比例 | 实际尺寸（参考） |
|------|------|-----------------|
| 头高 | 1 头 | 23-25 cm |
| 下巴到乳头 | 1 头 | 23-25 cm |
| 乳头到肚脐 | 1 头 | 23-25 cm |
| 肚脐到裆部 | 1 头 | 23-25 cm |
| 裆部到大腿中部 | 1 头 | 23-25 cm |
| 大腿中部到膝盖 | 1 头 | 23-25 cm |
| 膝盖到小腿中部 | 1 头 | 23-25 cm |
| 小腿中部到脚底 | 0.5 头 | 11-12 cm |

### 宽度比例

| 部位 | 比例 | 说明 |
|------|------|------|
| 肩宽 | 2-2.5 头 | 男性较宽 |
| 腰宽 | 1-1.2 头 | 男性较直 |
| 臀宽 | 1.2-1.5 头 | 女性较宽 |
| 臂展 | ≈身高 | 臂展约等于身高 |

## 🎨 Q 版风格（2-3 头身）

### 二头身

```
总身高：2 个头高
头高：占总身高 50%
特点：可爱、幼儿比例
```

**比例分配：**
- 头部：50%
- 躯干：25%
- 腿部：25%

### 三头身

```
总身高：3 个头高
头高：占总身高 33%
特点：卡通、少年比例
```

**比例分配：**
- 头部：33%
- 躯干：33%
- 腿部：33%

---

## 🦸 超级英雄风格（8-9 头身）

### 夸张比例

```
总身高：8-9 个头高
肩宽：3 个头宽
特点：健美、力量感
```

**特征：**
- 倒三角身材
- 发达的胸肌和肩膀
- 细腰
- 长腿

---

## 📏 面部比例

### 正视图

```
发际线到眉毛 = 1/3
眉毛到鼻底 = 1/3
鼻底到下巴 = 1/3

眼睛位置：头部 1/2 处
鼻子位置：眼睛到下巴 1/2 处
嘴巴位置：鼻子到下巴 1/3 处
```

### 侧视图

```
额头倾斜角：约 15-20 度
鼻子突出：约 1-2 cm
下巴突出：与额头齐平或略后
```

---

## 🤲 手部比例

### 成人手

```
手掌长度 ≈ 脸部长度的 2/3
手指长度 ≈ 手掌长度
中指最长，约 8-10 cm
```

### 简化画法

```
手掌：方形或梯形
手指：圆柱体
拇指：独立于其他四指
```

---

## 🦶 脚部比例

```
脚长 ≈ 头高
脚宽 ≈ 1/2 脚长
脚踝位置：内高外低
```

---

## ⚖️ 性别差异

### 男性

- 肩宽：2-2.5 头
- 腰臀比：接近 1:1
- 肌肉明显
- 重心偏高（胸部）

### 女性

- 肩宽：1.5-2 头
- 腰臀比：约 2:3
- 曲线明显
- 重心偏低（臀部）

---

## 🎯 建模检查清单

### 基础检查

- [ ] 头身比正确
- [ ] 左右对称
- [ ] 关节位置合理
- [ ] 比例符合风格

### 细节检查

- [ ] 眼睛在头部 1/2 处
- [ ] 手臂长度（指尖到大腿中部）
- [ ] 腿长比例
- [ ] 手脚大小

### 动态检查

- [ ] 关节活动范围
- [ ] 肌肉拉伸合理
- [ ] 服装不穿插
